# ecommerce-website
A modern, responsive E-Commerce website with product listings, category filters, sorting, dark mode, and a dynamic cart system. Built using HTML, CSS, and JavaScript.
## 🌐 Live Demo
[Click here to view the site](https://bbalasubramani.github.io/ecommerce-website/)

## 🛠️ Tech Stack
- HTML5
- CSS3 (Responsive design, Flexbox, Media Queries)
- JavaScript (DOM manipulation, Filtering, Sorting, Modal logic)
- Font Awesome for icons

## 🎯 Features
- 🌓 Dark Mode Toggle
- 🛒 Add to Cart with total price calculation
- 🧾 Category Filters & Rating Filters
- 🔄 Product Sorting (Price, Rating, Newest, Recommended)
- 💬 Toast notifications
- 📱 Fully responsive layout (Mobile-friendly)
- 🎨 Clean and modern UI (ShopNest branding)
